Track 3: Interactive prototypes

This is something I've actually grown to like doing. I like the process of prototyping. It really brings my ideas into action without needing to code everything. That is kinda the reason for prototyping. Figma is the first thing I think of when starting to design a new website. I do start on paper sometimes, but figma is much more handy for me. After Figma, I start to work with HTML and CSS and here things change a lot. The website starts to looks different and some of the design choices change. Sometimes for the good, but sometimes for the bad. For example, on my first portfolio website, the background was supposed to be a bit different, but I couldn't get the background image to be a certain way, so I left it like that. Overall, looking back, from the start of the semester to now, my knowledge of HTML and CSS has grown a lot and all I really need is more design ideas.
